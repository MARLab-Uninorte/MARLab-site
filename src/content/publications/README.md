# Publications

Add one `.yaml` file per publication. Example:

```yaml
title: "Título del paper"
authors: "Apellido, N., Apellido, M."
venue: "Nombre de journal/conferencia"
year: 2026
link: "https://doi.org/..."   # optional
tags: ["biomedical", "control"]  # optional
```

This file does not affect the site -- Astro's content loader only reads `.yaml` files here.
