# Open Source Software

Add one `.yaml` file per repository. Example:

```yaml
name: "nombre-del-repo"
description: "Qué hace la herramienta."
repoUrl: "https://github.com/org/repo"  # optional
tags: ["python", "ros2"]                 # optional
```

This file does not affect the site -- Astro's content loader only reads `.yaml` files here.
